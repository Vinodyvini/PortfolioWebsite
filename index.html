ToDo Application

This full-stack ToDo application enables efficient task management with CRUD operations and a comprehensive search functionality. Users can create, read, update, and delete tasks, and also filter tasks based on priority levels (high, medium, low) and categories like "work" and "personal" for better organization.

Key Features:

  • CRUD Operations: Implemented Create, Read, Update, and Delete functionality for task management.
  • Task Categorization: Organized tasks by priority (e.g., high, medium, low) and categories (e.g., work, personal) for better task classification and easy filtering.
  • Search Functionality: Enabled real-time search, allowing users to locate tasks quickly by keywords and filter criteria.
  • Error Handling: Integrated robust exception handling for user-friendly error messages and seamless user experience.

Technology Stack:

Backend: C# .NET Web API (.NET 8)
